--------------------------
Tests Folder
--------------------------

Contains notebooks that generate data for equation as follows:

y = -0.5x0 + 0.5x1 + 3

gplearn_test - contains code to create data (already in readable arrays), code for GPLearn symbolic regression
AI Feynman v2_test - contains code to create data (in .txt file), code for AI-Feynman 2.0 symbolic regression
PySR_test - contains code to create data (already in readable arrays), code for PySR symbolic regression
testeqn_benchmark - contains code for all 3 symbolic regressors (for runtime-specific tests)

Note: Code available to plot the data (matplotlib), code to create noisy data (introducing noise in data based on normal distribution)
